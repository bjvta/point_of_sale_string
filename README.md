# Point of sale problem

## Requirements
 - Ruby version 2.7.7
 - Bundler
 - RSpec

## Installation

Clone the project

`git clone asdfasdfa
cd asdfasdfs`

To install ruby using rbenv:
		`rbenv install 2.7.7`
		
To set the local in the project
	`rbenv local 2.7.7`

Install dependencies
`	gem install bundler:2.3.8
	bundle install`

## Running tests
`ruby test`


Happy testing!
